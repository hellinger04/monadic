# Lesson 3.2: Environment Monad

This is a possible implementation of the environment monad

```javascript
function ret(x) {
    return function (e) {
        return x;
    }
}

function bind(m, f) {
    return function (e) {
        return f(m(e))(e);
    }
}

function run(m) {
    return m($("window"));
}

function getenv(e) {
    return function () {
        return e.prompt();
    }
}
```
